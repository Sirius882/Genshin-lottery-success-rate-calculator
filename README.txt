武器接受度：
	0：池里另一把武器对自己完全没价值
	1：两把武器价值基本等同
	3：另一把武器也有用，但不如定轨的武器好
	暂不支持反向定轨玩法

抽取序列：
	根据你的抽卡优先级，c代表角色，w代表武器。
	例：想先抽2命火神，再抽0命茜特菈莉，有余力再抽两把武器+火神3命，则输入：ccccwwc

输出结果：
	一组圆括号-概率对表示一种可能出现的结果。圆括号第一个数表示达到的序列
	如下例，(6, 1): 0.16表示有16%的概率出6个角色+歪1把同池的up武器
